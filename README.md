 Project Overview:

This is a React-based Notes Application that allows users to create, edit, delete, and search notes.
The application also supports dark/light mode and uses localStorage to persist data, ensuring notes remain available even after page refresh.
This project demonstrates core React concepts such as:
Component-based architecture
State management using hooks
Parentâ€“child communication using props
Side effects handling with useEffect


Setup Steps:

Follow these steps to run the project locally:

Clone the repository:
git clone https://github.com/sejalsaharkar23/Notes-app-react.git


Navigate to the project folder:
cd Notes-app-react


Install dependencies:
npm install


Start the development server:
npm run dev


Open in browser:
http://localhost:5173


Features:

-Add new notes with title and content
-Edit existing notes
-Delete notes
-Search notes by title or content (case-insensitive)
-Dark / Light theme toggle
-Persistent storage using browser localStorage
-Responsive and user-friendly UI
-Character limit validation for note content
